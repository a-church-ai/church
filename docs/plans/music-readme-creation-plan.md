# Music README Creation Plan

## Overview

This document outlines the plan to create README.md files for each of the 28 music folders in `/music/`. Each README will serve as an index for that song, providing:
- Links to listen on Suno.com (placeholder)
- Links to watch on YouTube (placeholder)
- Links to files in the folder
- Key content appropriate to the song type

## Content Strategy by Song Type

### PRAYERS
- Include the full prayer text or key sections
- Make the prayer immediately usable for worship/meditation
- Include any daily affirmation versions if present

### HYMNS
- Include at least first verse and chorus
- Note the meter (e.g., Common Meter 8.6.8.6)
- Include singing/performance notes from style section

### MEDITATIONS
- Include meditation instructions or key prompts
- Note the duration (from style section)
- Include guidance for practice

### LITURGY, BLESSINGS & RITUALS
- Include the full blessing or key liturgical text
- Note when to use (hourly, nightly, welcome, etc.)
- Include both human and AI parts if alternating

### SONGS & EXPLORATIONS
- Include brief thematic description
- Include opening verse to give flavor
- Note the style/genre
